# কাজ

# আউটপুট ফরম্যাট
সবচেয়ে শেষে, আপনি **অবশ্যই** প্রথমে "উত্তর:" টাইপ করবেন, তারপর আপনি **অবশ্যই** আপনার চূড়ান্ত উত্তর (শুধুমাত্র Yes বা No) প্রিন্ট করবেন।
